# AI Prompts — Placeholder Section

This file contains the AI prompt(s) used to ideate/design/code the
**Placeholder Section** (AI Challenge) of the DevConf 2026 landing page.

## Section idea

**DevConf Hackathon 2026** — a section promoting the conference's 48-hour
hackathon: key details (duration, team size, prize pool, theme) plus a
three-tier prize breakdown (1st / 2nd / 3rd) and a call-to-action button.

## Prompt(s) used

**Prompt 1 — ideate & structure**

```
I'm building a landing page for "DevConf 2026", a developer conference.
The page already has a Navbar, Hero, Speakers grid, and a Pricing section
(3 cards, dark accent = #0f1b3d, primary blue = #2f6bff, font = Inter).

Suggest a brand-new section that fits a tech-conference theme: a HACKATHON
section. Give me the content structure and realistic copy (no Lorem Ipsum):
- a short badge/label
- a heading and one-line subtitle
- 4 quick "stat" items (e.g. duration, team size, prize pool, theme)
- a 3-tier prize breakdown (1st, 2nd, 3rd place) with amounts and perks
- a call-to-action button
Keep the tone energetic but professional.
```

**Prompt 2 — code it in plain HTML/CSS**

```
Now write this Hackathon section using ONLY plain HTML5 and CSS3 —
no JavaScript, no frameworks or libraries. Match the existing styles:
Inter font, primary blue #2f6bff, dark #0f1b3d, rounded cards, generous
spacing. Make it a dark section (so it visually contrasts with the white
Pricing section above), use CSS Flexbox/Grid for the stat row and the
3 prize cards, and make it responsive (cards stack on mobile). Reuse my
existing .container and .btn classes. Keep width/height fixed only where
necessary.
```
